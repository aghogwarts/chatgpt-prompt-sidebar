# Prompt sidebar inside ChatGPT
Allows you to view a sidebar on the right containing the prompt history of the focused chat. (Replicated a friend's idea which wasn't available for firefox)

Features:
- Seamless Sidebar integrated in the ChatGPT menu
- Highlight the current prompt integrated with scroll tracking
- Jump to the part of the chat by clicking the prompt

> [!WARNING]
> This add-on is experimental.
